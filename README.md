## ‚öôÔ∏è Backend Setup (Django)

Follow these steps to run the backend locally:

| Step | Command | Description |
|------|---------|-------------|
| Install requirements | `pip install -r requirements.txt` | Installs all necessary Python packages |
| Apply migrations | `python manage.py migrate` | Sets up the database tables |
| Run server | `python manage.py runserver` | Starts the Django development server |

Once the server is running, you can access the backend at:  
**http://127.0.0.1:8000/**
